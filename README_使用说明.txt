使用方法：
1. 解压 zip。
2. 把 index.html、game.js、images 文件夹一起上传到 GitHub Pages / Netlify 项目根目录。
3. 如果你之前把图片放在根目录，也可以继续放根目录；game.js 会按 根目录 → assets/ → images/ 的顺序查找图片。
4. 更新后请强制刷新浏览器，避免缓存旧 game.js。

版本变化：
- 从无限房间改为无限长廊。
- 开完一扇门后，当前门位横向滑走，新门位滑入。
- 保留鬼、人、Boss、鬼眼、图鉴、封印等设定。
